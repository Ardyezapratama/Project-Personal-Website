JAVASCRIPT

Javascript merupakan bahasa pemrograman yang wajib dipelajari jika ingin mendalami dunia web developtmet. Saat ini javascript tidak hanay digunakan di sisi client (browser) saja. Javascript juga digunakan pada server, console, program dekstip, mibile, IoT, game dan lain - lain.


Javascript awalnya bernama Mocha, lalu berubah menjadi LiveScript saat browser Netscape Navigator 2.0 rilis versi beta (september 1995). Namun, setelah itu dinami ulang menjadi Javascript.


Peralatan untuk Belajar Javascript


Peralatan yang harus disiapkan untuk belajar Javascript yaitu :

  • Web Browser (Google Chrome, Firefox, Opera dll
  • Teks Editor (Sublime Text, VS Code, dll

Penulisan Kide Javascript dengan Embed


Pada cara ini menggunakan tag "script" untuk menempelkan (embed) kode Javascript pada HTML. Tag ini dapat ditulis di dalam tag "head" dan "body".

Contoh :

<!DOCTYPE html> <html> <head> <title>Belajar Javascript dari Nol</title> <script> // ini adalah penulisan kode javascript // di dalam tag <head> console.log("Hello JS dari Head"); </script> </head> <body> <p>Tutorial Javascript untuk Pemula</p> <script> // ini adalah penulisan kode javascript // di dalam tag <body> console.log("Hello JS dari body"); </script> </body> </html>


Penulisan Kode Javascript Inline


Pada cara ini, kode javascript di letakkan di dalam atribut HTML. Cara ini biasanya digunakan untuk memanggil suatu fungsi pada event tertentu. Misalnya pada saat link diklik.

Contoh :

<a href="#" onclick="alert('Yey!')">Klik aku!</a> Atau bisa juga seperti ini : <a href="javascript:alert('Yey!')">Klik aku!</a>


Penulisan Kode Javascript External


Pada cara ini, penulisan kode javascript secara terpisah dari file HTML.
Cara ini biasayan digunakan pada proyek - proyek besar, karena diyakini dengan cara ini dapat lebih mudah mengelola kode project.

Contoh :
pertama buat terlebih dahulu file HTML dan Javascript.

belajar-js/
|-- kode-program.jsbr |-- index.html

Isi dari file kode-program.js

alert("Hello, ini adalah program JS eksternal!"); Isi dari file index.html <!DOCTYPE html> <html> <head> <title>Belajar Javascript dari Nol</title> </head> <body> <p>Tutorial Javascript untuk Pemula</p> <!-- Menyisipkan kode js eksternal --> <script src="kode-program.js"></script> </body> </html> Jadi caranya adalah dengan menyisipkan dengan memberikan atribut "src" pada tag "script". Maka, apapun yang ada di dalam file kode-program.js akan dapat dibaca dari file index.html.
