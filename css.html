CSS

CSS berfungsi untuk mengatur seluruh tampilan website sehingga terlihat lebih menarik dan sesuai dengan kebutuhan user.
CSS sangat berguna dalam proses pengembangan website, hampir setiap website sekarang menggunakan CSS sebagai tools untuk mengatur berbagai tampilan di dalamnya.


Pengertian CSS


CSS adalah kepanjangaan dari Cascading Style Sheets yang berguna untuk menyederhanakan proses pembuatan website dengan mengatur elemen yang tertulis di bahasa markup.


Ada banyak hal yang bisa dilakukan menggunakan CSS seperti mengatur warna teks, jenis font, baris antar paragraf, ukuran kolom, dan jenis background yang dipakai. CSS juga bisa dipakai untuk mendesign layout, variasi tampilan di berbagai perangkat yang berbeda, dan berbagai efek yang dipakai di dalam website.



Fungsi CSS

1. Mempercepat Proses Desain

Jika menggunakan CSS cukup mengetikkan satu kali fungsi CSS kemudian menggunakannya diberbagai halaman HTML tanpa harus menyalinya satu persatu. Fungsi CSS yang dibuat dalam satu file dapat dipanggil ke berbagai halaman web tanpa harus menyalin baris kode fungsi berkali-kali.



2. Halaman Cepat Dimuat

Jika menggunakan CSS tidak perlu menuliskan atribut tag HTML di setiap file, hanya cukup menuliskan satu aturan CSS dan menerapkannya di berbagai file yang dibutuhkan hanya dengan memanggilnya.


Jadi satu file hanya terdiri dari sedikit baris kode, inilah yang membuat proses download menjadi lebih cepat.



3. Proses Pemeliharaan Lebih Mudah

CSS memudahkan untuk mengubah tampilan di berbagai halaman. Hanya dengan mengubah fungsi style di file CSS maka seluruh tampilan yang menggunakan fungsi tersebut akan berubah secara otomatis.



4. Style Lebih Beragam Dibanding degan HTML

CSS mempunyai atribut lebih beragam dibandingkan dengan HTML, dan mempunyai lebih banyak pilihan tampilan halaman website.



5. Kompatible Dengan Berbagai Macam Perangkat

CSS memungkinkan konten dapat dioptimasi dilebih dari satu perangkat. Misal ketika memproses sebuah dokumen. Jika menggunakan CSS, bisa menyesuaikan tampilan dokumen di perangkat versi lama sekaligus di versi yang baru.



6. CSS Menjadi Standar Pengembangan Website

Hampir seluruh website yang ada di internet menggunakan CSS di dalamnya. Selain tampilannya yang lebih menarik, kebanyakan browser populer saat ini juga mendukung CSS. Jadi jika ingin mengembangkan website yang sesuai standar browser populer, sebaiknya menggunakan CSS.



3 Macam CSS di Dalam HTML

Pada CSS terdapat 3 jenis penulisan kode didalam HTML yaitu External, Internal dan Inline.



External

External CSS adalah penempatan baris kode CSS menggunakan file .css. Cara ini dapat digunakan untuk mendeklarasikan sebagian besar pengaturan tampilan website secara keseluruhan. Cara ini lebih sederhana dan simpel daripada menambahkan baris kode di setiap elemen HTML yang ingin Anda atur tampilannya.


Jika menggunakan penempatan secara eksternal, Anda harus menambahkan header untuk memanggil file .css di dalam file HTML seperti contoh di bawah ini.

<head> <link rel="stylesheet" type="text/css" href="style.css"> </head>


Baris kode di atas akan menghubungkan file .html dengan eksternal style sheet (CSS). Pada contoh di atas file CSS menggunakan nama ‘style.css’ yang dapat Anda sesuaikan dengan letak file CSS di dalam direktori website. Ketika file HTML sudah terhubung dengan file CSS maka seluruh pengaturan CSS akan diterapkan di dalam kode HTML.



Internal

Penggunaan CSS secara internal yaitu dengan cara menuliskan kode di bagian header HTML. cara ini sangat cocok untuk dipakai ketika membuat halaman web dengan tampilan yang berbeda. Contoh CSS secara internal seprti di bawah ini.

<head> <style type="text/css"> body{ background-color: red; } </style> </head>

Baris kode di atas akan menerapkan background dengan warna merah pada bagian body HTML.



Inline

Inline CSS adalah penempatan CSS yang langsung dilakukan di dalam konten HTML. Penempatan CSS secara inline hanya akan mempengaruhi satu bagian baris kode. Berikut adalah contohnya.

<p style="color:red; font-size: 20px;">Ini adalah paragraf</p>

CSS di atas berarti hanya akan mempengaruhi paragraf yang spesifik di dalam satu file .html dan muncul dengan warna merah (merah) dan ukuran font 30px.
